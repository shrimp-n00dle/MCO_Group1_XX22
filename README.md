# Garnet

    This web application, Garnet : Hidden Gem for Gamers!, Garnet for short, will be a space for game developers of all roles to connect, find and converse with like-minded people, as well as simply share their projects with other developers and find other projects to be inspired by. 

GROUP MEMBERS:
    - Montuerto, Yzabelle Anne
    - Sadio, Megan Florence Sophia
    - Vingno, Jan Elizabeth

This project uses Node.js and handlebars.
    - To run, enter "node app.js" in the terminal.
    - Running "localhost:3000/" or "localhost:3000/welcome" in a browser will then allow you to experience the site.
